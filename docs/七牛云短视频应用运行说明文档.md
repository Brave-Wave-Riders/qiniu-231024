# 七牛云短视频应用运行说明文档

## 前端

node版本： v14.17.0

```bash
cd qiniu-231024/front/qiniu-video-web
npm i
npm run serve
```

## 后端

golang版本： v1.19

```bash
cd qiniu-231024/backend
go mod tidy
# run之前确保依赖服务如mysql已经启动
go run main.go
```

`注意：`conf/config.yaml文件为配置文件，需填写mysql，七牛云key，程序port等配置。
